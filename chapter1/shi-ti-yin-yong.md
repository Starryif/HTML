# 实体引用： 在HTML中包含特殊字符

在HTML中，字符 &lt;, &gt;,",' 和 & 是特殊字符. 它们是HTML语法自身的一部分, 那么如何将这些字符包含进文本中呢, 比如说如果真的想要在文本中使用符号&或者小于号, 而不想让它们被浏览器视为代码并被解释? 必须使用字符引用 —— 表示字符的特殊编码, 它们可以在那些情况下使用. 每个字符引用以符号&开始, 以分号\(;\)结束.

| 原义字符 | 等价字符引用 |
| :--- | :--- |
| &lt; | &lt; |
| &gt; | &gt; |
| " | " |
| ' | ' |
| & | & |
